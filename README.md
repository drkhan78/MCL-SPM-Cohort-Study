# MCL-SPM-Cohort-Study
Matched cohort study analyzing colorectal cancer risk in Mantle Cell Lymphoma patients using R

## Data Availability
The datasets used in this analysis (`Demographic_data.txt` and `Register_data.txt`) are proprietary materials provided for a research assessment. To comply with data privacy and institutional intellectual property policies, the raw data files are not included in this repository. The provided R scripts contain the logic used to process these datasets and can be adapted for similar data structures.
